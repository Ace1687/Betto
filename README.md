# Betto 🤖

Betto is a custom desktop AI assistant built in Python.  
It combines chat, voice activation, and automation into a single app designed to feel personal and interactive.

---

## ✨ Features

- 💬 Chat interface (send messages directly to the AI)
- 🎙️ Voice activation (hotkey: Ctrl + B)
- 🗣️ Optional wake word support ("Hey Betto")
- 🛠️ Tool system (create files, search the web, automate tasks)
- 🖥️ Desktop app UI (PyQt6-based)

---

## 🧠 How it works

Betto connects to an AI agent and can:
- respond to messages
- execute tools (like creating files or searching)
- interact through both text and voice
